# 단타 전략 텔레그램 봇

## 🎯 프로젝트 설계 원칙

### 필수 고려사항
1. **대화형 플로(Conversational Flow)** - 모든 명령어는 단계별 대화 방식
2. **간편한 UX** - 최소한의 입력으로 최대 기능 수행
3. **파라미터 순서** - Help Guide 샘플 제시, 쉼표로 구분된 순차 입력

## ✨ 주요 기능

### Tactic1: 급상승 첫 조정 매매
- 당일 시가 전날 종가 대비 7%↑ 자동 탐지
- 1분봉 기준 첫 조정 후 반등 지점 매수
- 부분 익절 지원

### Tactic2: 스윙 분할 매수
- 지지선에서 1차/2차 분할 매수
- 조건부 자동 익절

### 가격 모니터링
- `.env`의 `MONITOR_INTERVAL`로 체크 주기 설정 (초 단위)
- Tactic1/2 감시 리스트 자동 모니터링
- 텔레그램 시그널 알림

## 🌐 웹 UI (다중 페이지)

### 5개 페이지 구조
1. **📊 감시리스트**: Mode1 + Mode2 통합 뷰, 필터링 (Mode/상태/날짜)
2. **📈 Mode1**: 전일대비 급등 첫 조정 (분봉 기반, 그린라이트 알림) - 완료 ✨
3. **📉 Mode2**: 저항/지지 레벨 자동매매 (완료)
4. **💰 매매일지**: 확정 손익 내역, 통계 - 구현 대기
5. **🧪 Test**: Kiwoom API 테스트 (종목 정보, 일봉차트, 토큰 상태)

### Mode1 주요 기능
1. **종목 등록**: 종목코드 입력 (종목명 자동 조회)
2. **모니터링 조건 설정**: 
   - 분봉 종류 선택 (1/3/5/10분)
   - 추세 선택 (상승/하락)
   - 연속 봉 개수 지정 (예: 1분봉 4개 연속 양봉)
   - 조회할 봉 개수 지정 (최대 200개)
   - 여러 조건 동적 추가/삭제 가능
3. **그린라이트 시스템**: 
   - 모든 조건이 AND로 만족되면 알림
   - 텔레그램으로 종목명, 모니터링가, 조건 상태 전송
   - 최근 분봉 요약 (양봉/음봉 시각화)
4. **시간 기반 Polling**:
   - 1분봉: 사용자 설정 (기본 20초)
   - 3분봉: 00:03, 03:03, 06:03, ... (3분 간격)
   - 5분봉: 00:03, 05:03, 10:03, ... (5분 간격)
   - 10분봉: 00:03, 10:03, 20:03, ... (10분 간격)
5. **기대 수익률 설정**: 목표 수익률 입력
6. **인사이트 실시간 표시**: 각 조건별 만족 여부 + 최근 분봉 상태
7. **Status**: 매수대기 → 매도대기 → 자동매도/수동매도

### Mode2 주요 기능
1. **종목 등록**: 차트 기반 입력 (종목코드 + Budget + 저항/지지 레벨)
   - 종목코드 입력 시 종목명 자동 조회
   - Budget 입력으로 매수 수량 자동 계산
   - Polling 주기 선택 (5/10/15/20/30초)
   - **🔔 알림만 체크박스**: 자동매매 없이 알림만 받기
2. **체크박스 선택**: Row 클릭 시 폼에 데이터 auto-fill
3. **인라인 에디팅**: 테이블에서 직접 수정 가능
   - 종목명, 매수타점, Budget, Polling 주기
   - 저항/지지 레벨 (가격 + 익절/손절 %)
   - 모드 태그 클릭으로 알림/자동 전환
4. **자동 계산**: 매수 수량 = Budget / 매수타점
5. **2가지 모드**:
   - **🔔 알림 전용**: 매수타점 도달 시 텔레그램 알림만
   - **🤖 자동매매**: 알림 + 자동 주문 실행
     - 매수: 매수 타점 (±1%) 도달 시
     - 익절: 1차/2차 저항 가격 도달 시
     - 손절: 1차/2차 지지 가격 하락 시
6. **Polling 최적화**: 종목별 주기 설정으로 API 효율화

### Test 페이지 기능
- **종목 정보 조회**: 종목코드 → 종목명, 현재가
- **일봉차트 조회**: 종목명/코드 → 당일/전일 가격 정보 (롯데케미칼 예시)
- **분봉 차트 조회**: 1/3/5/10분봉
- **토큰 상태 확인**: Kiwoom API 연결 상태
- **⚠️ 실제 주문 테스트** (주의!)
  - 매수 주문: 종목코드, 수량, 시장가/지정가, 가격
  - 매도 주문: 종목코드, 수량(비우면 전량), 시장가/지정가, 가격
  - 시뮬레이션 모드: `.env`에 `ORDER_SIMULATION_MODE=1` 설정
