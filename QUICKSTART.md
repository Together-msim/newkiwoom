## 🎯 이제 무엇을 하나요?

### 옵션 A: 알림만 받기 (안전)

**현재 상태 유지**:
- `ORDER_SIMULATION_MODE=1` (시뮬레이션)
- 🔔 알림만 체크됨

**동작**:
1. 매수타점 도달 → 텔레그램 알림 📱
2. 수동으로 HTS/MTS에서 매수 결정
3. 수동으로 매도 결정

**장점**: 
- ✅ 100% 안전
- ✅ 시스템 학습 가능
- ✅ 판단은 내가 함

### 옵션 B: 자동매매 시작 (경험자)

#### ⚠️ 주의: 실제 주문이 실행됩니다!

**1. .env 파일 수정**
```bash
# 에디터로 .env 파일 열기
nano .env

# 또는
code .env

# 수정 내용:

# 저장 후 종료
```

**2. 서버 재시작**
```bash

python web_app.py
```

**3. Mode2에서 자동매매 종목 등록**
```
1. Mode2 페이지
2. 종목 입력
3. ⚠️ 알림만 체크박스 해제 (자동매매 모드)
4. 저장

결과: 🤖 자동 태그 표시
```

**4. 모니터링**
```
- 웹 UI: 감시리스트 확인
```

## 🛠️ 자주 쓰는 기능

### 1. 보유종목 조회
```
📊 감시리스트 페이지
→ 💼 보유 종목 조회 버튼
→ 모달에서 확인
```

### 2. 보유종목 매도
```
보유종목 모달에서
→ 매도 버튼 클릭
→ 수량/가격 입력
→ 매도 실행
```

### 3. 보유수량 동기화
```
📊 감시리스트 페이지
→ 🔄 보유수량 동기화 버튼
→ 감시리스트 업데이트
```

### 4. 모드 전환 (알림 ↔ 자동)
```
Mode2 테이블에서
→ 모드 태그 클릭
→ 🔔 알림 ↔ 🤖 자동
```

### 5. 종목 수정
```
Mode2 테이블에서
→ 체크박스 클릭
→ 폼에 자동 채워짐
→ 수정 후 저장
```

## 💡 실전 팁

### 초보자 전략
```yaml
종목: 1개 (대형주)
Budget: 10만원
모드: 알림 전용
Polling: 10초
기간: 1주일 (시스템 학습)
```

### 진행 순서
```
1주차: 알림 전용 + 시뮬레이션
2주차: 알림 전용 + 수동 매매
3주차: 자동매매 + 소액 (10만원)
4주차: 자동매매 + 증액 (50만원)
```

### 안전 수칙
```
1. 항상 손절선 설정
2. Budget 제한 (종목당 최대 100만원)
3. 대형주 위주 (변동성 낮음)
4. 분산 투자 (5~10 종목)
5. 텔레그램 알림 항상 확인
```

## ❓ FAQ

### Q: 지금 당장 실전 가능한가요?
A: 네! **알림 전용 모드**로 바로 시작하세요.
- `ORDER_SIMULATION_MODE=1` 유지
- 🔔 알림만 체크박스 체크
- 알림 받고 수동 매매

### Q: 자동매매는 언제 시작하나요?
A: 시스템에 익숙해진 후 (1~2주 후)
- 알림 정확도 확인
- 매수/매도 타이밍 학습
- 소액으로 테스트 (10만원)

### Q: 손실 위험은?
A: Budget으로 제한됩니다
- 종목당 Budget 설정 (예: 10만원)
- 손절선 설정 (예: -5%)
- 최대 손실 = Budget × 손절선%

### Q: 여러 종목 동시에?
A: 가능합니다
- Mode1: 분봉 조건 기반
- Mode2: 저항/지지 레벨 기반
- 각각 독립적으로 모니터링

### Q: 장외 시간에도 작동하나요?
A: 제한적으로 작동
- 모니터링: 계속 작동
- API 호출: 실패 가능
- 주문: 장 시간에만 가능
